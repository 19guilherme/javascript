Todas as Variáveis tem que ter um nome

"=" significa RECEBE

"null" signica NULO, VARIAVEL SEM VALOR.

Colocar valor nas Variaveis CHAMA-SE DE "ATRIBUIÇÃO"

Variaveis

Variaveis NUMBER

5, 18, -12 = Números inteiros

0.5, -15.9, 3.14, 8.0 = Números Reais

  • var n1=5
  • var n2=8,5
  • var n3=4,5
  • var n1=6
  • var n2=158
  • var n3=1000
-----------

Variavel STRING

Por definição, strings são sequências de caracteres alfanuméricos (letras, números e/ou símbolos), pode-se ser criadas usando as Aspas ("Aspas Duplas", 'Aspas Simples ou apóstrofo', `Crase`)

  • var s1= "javascript"
  • var s2='Curso em vídeo'
  • var s3=`Guanabara`
-----------

Variavel BOOLEAN

True e false

Colocando nome nas Variáveis

  • Podem começar com letra, $ ou _
  • Não pode começar com números
  • É possível usar letras ou números
  • É possível usar acentos e simbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Dicas e Observação

  • Maiúscula e minúscula fazem diferença
  • Tente escolher "nomes coerentes"
    (salario=salario / idade=idade / nome=nome / sexo=sexo)
  • Evite se tronar um programador alfabeto(a, b, c, d, e, ...) ou um programador contador(1, 2, 3, 4, ...)
