Somando valores no window.prompt

var n1 = window.prompt('Digite um número: ') //STRING
/* window.prompt RECEBE os valores como STRING */
var n2 = window.prompt('Digite outro número: ') //STRING
var s = n1 + n2

//(Number + Number) para Adição
//(String + String) para Concatenação

Conversão de STRING para NUMBER

  • Number.parseInt(n) - (número inteiro 5, -13, 27, ...)
  • Number.parseFloat(n) - (número real 3.14, 5.07, -13.97, ...)
  • Number(n) - O proprio JS fica Responsavel por escolher real ou inteiro.

Conversão de NUMBER para STRING

  • String(n)
  • n.toString()

Formantando Strings

var s = 'JavaScript'

  • s.length // quantos caracteres a string tem
  • s.toUpperCase() // TUDO para "MAIÚSCULA"
  • s.toLowerCase() // TUDO para "minúscula"

'Eu estou aprendendo s' // Não faz Interpolação
'Eu estou aprendendo ' + s // Usa Concatenação
`Eu estou aprendendo ${s}` // usa Template string (Usar CRASE AO INVES DE ASPAS).

Exemplo:

  • nome = Guilherme
  • idade = 26
  • nota = 7.8

'eu ' + nome + 'tenho ' + idade + 'anos e tirei uma nota de ' + nota // Forma antiga, mas, ainda aceitavel

`eu ${nome} tenho ${idade} anos, e tirei uma nota de ${nota}` // Forma mais atual de se escrever

Resultado: eu Guilherme tenho 26 anos, e tirei uma nota de 7.8
